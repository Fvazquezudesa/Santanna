## Key findings (confirmed by agents)

### BUG 1 — CRITICAL: 555 missing winners
- PDF has 1,010 winners ("N° de Orden Asignado"), DTA has only 453 (`ganador_oct=1`).

- Additional 9 winners hidden inside 447 typo-recovered CUILs (bug 3).
- 1 winner found only in PDF (not in DTA even via typo recovery).
- **Impact:** Using `ganador_oct` as treatment indicator would miss ~55% of true winners for this sorteo.

### BUG 2: 2 duplicated cupo texts
CUILs `20258317875` and `27297654778` have `cupo = "Sin Cotitular con Hijos/as Sin Cotitular con Hijos/as"` (double). PDF has single. Agent 7 confirmed it's a DTA concatenation bug.

### BUG 3: 447 typo CUILs
- 470 CUILs "only in DTA"; 95.1% (447) have a PDF counterpart within same DU+tip+cupo at edit distance ≤ 4.
- 211 have non-11-digit length (likely Stata numeric ingestion stripping leading zeros).
- 134 pure digit transpositions; 210 edit-distance=1; 78 ed=2; 22 ed=3-4.
- 23 DTA residuals + 80 PDF residuals unmatched.

- **Tipología**: 0 real discrepancies (509 apparent mismatches were my PDF parser issue — MN1A/MNC1 detection — confirmed by agent 7).
